# The Business Intelligence Automator

## Description
This tool automates the processing of daily sales logs. It extracts data from multiple Excel workbooks, calculates KPIs, and generates a professional weekly executive summary including:
*   **Total Revenue** and **Average Daily Sales**.
*   **Best Performing Day** detection.
*   **Most Sold Item** analysis.
*   **Comparative Layout** of item quantities across the week.
